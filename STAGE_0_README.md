# AI-Based Crime Detection System for Supermarkets - Stage 0

## Project Overview
An intelligent CCTV surveillance system that uses AI/ML to detect suspicious and theft-related behavior in real-time at supermarkets. The system provides automated alerts to shop owners and stores video evidence for investigation.

## Stage 0: Foundation & Planning

This is the initial stage that includes:
- Project structure and organization
- Basic HTML UI framework
- Project documentation
- Requirements and dependencies
- System architecture overview

## Key Features (To Be Implemented)
- **Live Video Monitoring**: Real-time CCTV feed processing
- **AI-Powered Detection**: Suspicious behavior detection using ML models
- **Automated Alerts**: Instant notifications to shop owners
- **Video Evidence**: Recording and storage of theft incidents
- **Dashboard**: Visual interface for monitoring and alerts

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Python (to be added in next stages)
- **AI/ML**: YOLO (object detection), TensorFlow (behavior recognition)
- **Video Processing**: OpenCV
- **Notifications**: Email/SMS alerts

## Next Steps (Stage 1 and Beyond)
1. **Stage 1**: Backend setup with Flask/Django
2. **Stage 2**: CCTV integration and video processing
3. **Stage 3**: AI model integration (YOLO + TensorFlow)
4. **Stage 4**: Alert system implementation
5. **Stage 5**: Testing and deployment

## System Requirements
- Python 3.8+
- OpenCV 4.5+
- TensorFlow 2.x
- YOLO v5/v8
- CCTV cameras or USB webcams
- Processing device with GPU support (recommended)
- 8GB+ RAM
- Storage for video evidence (minimum 100GB recommended)
